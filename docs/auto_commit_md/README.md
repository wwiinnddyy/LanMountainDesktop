## 查看完整提交历史

如需查看完整的提交历史，请使用以下命令：

```bash
# 查看所有提交
git log --oneline

# 查看详细提交信息
git log --pretty=format:"%H|%an|%ad|%s" --date=format:"%Y-%m-%d %H:%M:%S"

# 查看特定提交的详细变更
git show <commit_hash>
```

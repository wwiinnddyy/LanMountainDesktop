## 🔧 技术栈

- **.NET 10** - 应用框架
- **Avalonia UI 12** - 跨平台 UI 框架
- **FluentAvalonia** - Fluent Design 控件库
- **CommunityToolkit.Mvvm** - MVVM 框架
- **PLONDS** - 应用增量更新与分发系统
- **dotnetCampus.Ipc** - 进程间通信

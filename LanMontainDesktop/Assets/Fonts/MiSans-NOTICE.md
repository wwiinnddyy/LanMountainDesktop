# MiSans Font Notice

This app bundles MiSans fonts for consistent cross-device rendering.

## Included files

- `MiSans-Regular.ttf`
- `MiSans-Semibold.ttf`
- `MiSans-Bold.ttf`

## Source

- Upstream package repository: https://github.com/dsrkafuu/misans
- Original font source referenced by upstream: https://hyperos.mi.com/font/zh/
